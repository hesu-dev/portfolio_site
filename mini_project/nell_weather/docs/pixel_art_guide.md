# 픽셀 아트 캐릭터 & 스프라이트 제작 가이드

## 개요
Nell Weather 앱의 "아바타 메이커"와 "날씨 반응형 캐릭터" 기능을 구현하기 위한 그래픽 리소스 제작 가이드라인입니다.
이 가이드에 맞춰 이미지를 제작해주시면, 앱 내에서 자연스러운 애니메이션과 커스터마이징 기능을 구현할 수 있습니다.

## 1. 캔버스 규격 (Canvas Size)
가장 추천하는 사이즈는 **32x32px**입니다. (레트로한 느낌)
디테일을 살리고 싶다면 **64x64px**도 좋습니다.

*   **기본 캔버스**: 64x64px (추천)
*   **스프라이트 시트**: 여러 동작을 가로로 연결한 이미지

---

## 2. 파츠 분리 (Layer Strategy)
아바타 메이커(커스터마이징) 기능을 위해 캐릭터를 하나의 통이미지가 아니라, **부위별로 투명 배경(PNG)** 파일로 저장해야 합니다.

모든 파츠는 **동일한 캔버스 크기(예: 64x64)**를 유지해야 하며, 겹쳤을 때 위치가 정확히 맞아야 합니다.

### 필수 파츠 목록
1.  **Body (몸체)**
    *   얼굴형, 팔, 다리가 포함된 기본 베이스.
    *   피부색 변경을 위해 채색된 버전별로 필요 (또는 흰색 베이스에 코드에서 색상 입히기 가능)
2.  **Face (표정)**
    *   눈, 코, 입.
    *   다양한 표정 (웃음, 멍때림, 윙크 등)
3.  **Hair_Back (뒷머리)**
    *   몸체 뒤쪽에 그려지는 머리카락.
4.  **Outfit (의상)**
    *   몸체 위에 덧씌워질 옷. (상의, 하의 또는 한벌옷)
5.  **Hair_Front (앞머리)**
    *   얼굴과 몸체 가장 앞쪽에 그려지는 머리카락.
6.  **Accessory (소품 - 선택)**
    *   우산, 모자, 안경 등.

---

## 3. 애니메이션 스프라이트 (Animation Sprite Sheet)
각 파츠별로 움직임을 표현하기 위해 **프레임(Frame)** 단위로 이미지가 필요합니다.

### A. 기본 상태 (Idle) - 숨쉬기
캐릭터가 제자리에 서서 살짝 숨을 쉬는 듯한 모션입니다.
*   **프레임 수**: 4 프레임 (권장)
*   **동작**:
    *   Frame 1: 기본
    *   Frame 2: 1픽셀 아래로 (살짝 눌림)
    *   Frame 3: 기본
    *   Frame 4: 1픽셀 위로 (살짝 들림)

### B. 걷기 (Walk)
캐릭터가 걷는 모습입니다. (보통 측면이나 반측면)
*   **프레임 수**: 4 프레임
*   **동작**: 
    *   Frame 1: 오른발 앞으로
    *   Frame 2: 차렷 (중간)
    *   Frame 3: 왼발 앞으로
    *   Frame 4: 차렷 (중간)

### C. 날씨별 특수 동작 (Special Actions)
*   **우산 쓰기 (비)**: 팔을 들어 우산을 잡고 있는 포즈.
*   **떨기 (눈/추위)**: 몸을 웅크리고 부들부들 떠는 포즈.

---

## 4. 파일 전달 가이드

이미지를 전달해주실 때 아래 두 가지 방법 중 하나를 선택해주세요.

### 방법 1: 파츠별 스프라이트 시트 (추천)
각 파츠의 모든 프레임을 하나의 긴 이미지로 연결합니다.
*   예: `hair_long_walk.png` (64x64 이미지 4장이 가로로 붙어있음 -> 256x64 크기)
*   장점: 코드로 잘라서 애니메이션 돌리기 매우 쉽습니다.

### 방법 2: 낱장 이미지
*   폴더 정리가 중요합니다.
    *   `/hair/style1/idle_0.png`
    *   `/hair/style1/idle_1.png`
    *   `/body/pale/walk_0.png` ...

---

## 5. 요약 (Checklist)
네가 그려줘야 할 이미지는 다음과 같습니다.

1.  **[Body]**: 기본 몸체 (Idle 4장, Walk 4장)
2.  **[Face]**: 표정 아이콘 (애니메이션 불필요, 눈 깜빡임 정도만 있어도 됨)
3.  **[Hair]**: 헤어 스타일 N종 (Idle 4장, Walk 4장 - 머리 흔들림 포함)
4.  **[Outfit]**: 의상 N종 (Idle 4장, Walk 4장 - 옷주름 움직임 포함)
5.  **[Accessory]**: 우산 등 (손에 쥐는 위치 맞춰서)
